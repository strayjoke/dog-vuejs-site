# dog-vuejs-site

## 克隆项目
```
git clone git@github.com:strayjoke/dog-vuejs-site.git
```
>`dog-vuejs` 项目地址:[https://github.com/strayjoke/dog-vuejs](https://github.com/strayjoke/dog-vuejs)

## 全局安装 vuepress
```
yarn global add vuepress # 或者：npm install -g vuepress
```

## 开发模式
```
vuepress dev docs
```

## 生产模式
```
vuepress build docs
```
